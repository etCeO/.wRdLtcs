# Wordlytics
### Word Frequency Counter

- A Python program that reads .txt files and calculates word frequencies. It cleans and processes text input, counts occurrences using dictionaries, and outputs a sorted list of word counts. Useful for basic text analysis and learning core Python concepts like file handling and data structures.
